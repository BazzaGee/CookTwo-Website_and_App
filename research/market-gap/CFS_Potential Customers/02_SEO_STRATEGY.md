## 9. PWA-Specific SEO Considerations (From SEO Agent)

### Crawlability
- Ensure all pages are server-side rendered (SSR) or use prerendering for search bots
- Implement proper `<meta>` tags, canonical URLs, and structured data
- Create a comprehensive sitemap.xml
- Use Google Search Console to submit and monitor indexing

### Performance
- Target Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- Implement service worker for offline support (but ensure crawlable content without JS)
- Use semantic HTML throughout

### App-Like SEO
- Add `manifest.json` with proper `name`, `short_name`, `description`, `start_url`
- Implement `<link rel="manifest">` in HTML head
- Add Apple-specific meta tags for iOS PWA support
- Create app install banner triggers

### E-E-A-T Signals (Experience, Expertise, Authoritativeness, Trustworthiness)
- Create an "About" page with founder story and credentials
- Add author bios to blog posts with nutrition/fitness expertise
- Link to credible sources (studies, surveys) in all blog content
- Display trust signals: user testimonials, waitlist count, press mentions
- Register with Google Business Profile
